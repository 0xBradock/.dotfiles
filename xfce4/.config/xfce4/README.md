# XFCE4

XFCE is a lightweight desktop environment; this folder stores selected terminal and file-manager settings.

More information: https://docs.xfce.org/

## Contents

- `.config/xfce4/terminal/` - XFCE Terminal preferences and accelerators.
- `.config/xfce4/xfconf/xfce-perchannel-xml/` - exported XFCE settings such as Thunar and terminal channels.
