# i3 window manager

i3 is a tiling window manager for X11 focused on keyboard-driven workflows.

More information: https://i3wm.org/docs/

## Contents

- `.config/i3/config` - main i3 configuration.
- `.config/i3/keybindings` - extracted keybinding reference.
- `.config/i3/i3blocks.conf` - status bar block configuration.
- `.config/i3/scripts/` - helper scripts for status, power, display, weather, and launcher actions.
