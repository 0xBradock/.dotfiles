# rofi

rofi is an application launcher, dmenu replacement, and menu framework for X11.

More information: https://github.com/davatorium/rofi

## Contents

- `.config/rofi/config.rasi` - main rofi theme and behavior.
- `.config/rofi/powermenu.rasi` - power menu theme.
- `.config/rofi/power-profiles.rasi` - power profile menu theme.
- `.config/rofi/rofidmenu.rasi` - dmenu-style theme.
- `.config/rofi/rofikeyhint.rasi` - key hint theme.
